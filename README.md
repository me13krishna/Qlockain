# Qlockain

A simple Flask-based document verification and blockchain demo project.

## Quick start

1. Create and activate your virtualenv:

```powershell
python -m venv venv
& venv\Scripts\Activate.ps1
pip install -r requirements.txt
```

2. Run the app:

```powershell
python run.py
```

3. Open http://127.0.0.1:5000 in your browser.
